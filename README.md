# 📖 About

**HotelFinder** is a modern hotel discovery platform built with **React.js** that enables users to search, browse, and explore hotels through a REST API.

The application focuses on clean UI/UX, responsive layouts, reusable React components, and modern frontend development practices.

This project is being developed as part of a frontend internship assignment.

# 🚀 Getting Started

## Clone the Repository

```bash
git clone https://github.com/aryamantomar/HotelFinder.git
```

## Navigate to the Project

```bash
cd HotelFinder
```

## Install Dependencies

```bash
npm install
```

## Run Development Server

```bash
npm run dev
```

The application will start on:

```text
http://localhost:5173/
```
